# 📚 StudyNook – Room Booking Platform

StudyNook is a full-stack web application where students and library users can list, browse, and book study rooms. Users can also manage their own room listings and view/manage their bookings with real-time authentication and secure protected routes.

---

## 🌐 Live Demo

👉 https://study-nook-client-eight.vercel.app/

---

## 🛠️ Technologies Used

### Frontend

- Next.js (App Router)
- React
- Tailwind CSS
- HeroUI
- Framer Motion
- React Hot Toast
- React Icons

### Backend

- Node.js / Express.js
- MongoDB
- Better Auth (JWT-based authentication)
- MongoDB Native Driver

---

## 🚀 Features

- 🔐 Authentication (Email/Password + Google OAuth)
- 🏠 Create, edit, and delete study rooms
- 🔍 Search and filter rooms (price, amenities, name)
- 📅 Book rooms with time slot selection
- ⛔ Conflict-free booking system
- 👤 User dashboard (My Listings & My Bookings)
- 🧾 Role-based access protection
- 📱 Fully responsive UI (mobile-first)
- ⚡ Fast UI with Next.js App Router

---

## 🔐 Authentication System

- Built with **Better Auth**
